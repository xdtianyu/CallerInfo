##功能

1\. 查询来电号码并显示悬浮窗，显示位置和其他号码信息。

2\. 主界面显示最近来电列表。

3\. 在主界面可以查询电话号码信息。

4\. 在设置界面可以自定义设置卡片及悬浮窗的颜色。

5\. 自定义 API 密钥。

6\. 自定义悬浮窗, 如文字大小、透明度。
